## 📦 Project 1.0: Omniverse Extension Structure
This repository is now structured as a **NVIDIA Omniverse Kit Extension**.

### Folder Architecture
- `ext/`: Root extension directory.
- `config/extension.toml`: Metadata and dependency configuration.
- `quantum/omni/tool/`: Core Python logic for procedural generation.

### Capabilities
- **Batch Processing:** Can be called via the Kit Extension Manager.
- **Physics Integration:** Native hooks for `omni.physx`.
- **Quantum Simulation:** Automated state collapse measurement for large-scale digital twins.
